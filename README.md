## Install

01- Clone this repository:

```bash
$ git clone https://github.com/thiiagoms/url-short.git
```

02 - Change to repository directory:
```bash
$ cd url-short
```

03 - Execute `setup.sh`: 
```bash
url-short $ chmod +x ./setup.sh
```
